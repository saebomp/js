Determine if the function received the number of parameters it expected?
f(1);
f(1,2);
f(1,2,3);

let f = function (a, b) {

}

Correct Answer :

let f = function (a, b) {

if(arguments.length === f.length) { console.log('We have a match') } else { console.log('No match') }

}

-length is the property of the function that tells you how many things have been defined.
-arguments is the array like structure of all the elements that have been passed and all the parameters that have been passed in to the function.
