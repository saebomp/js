Async - Question 2
What is the resulting output?

let num = 0;

async function increment(){ num += await 2; //num = num + await 2; // Promise.resolve(2) console.log(num); }

increment();

num += 1;

console.log(num);

My Answer :

2 -> 3

the function increment will wait until num become 2. when the num become 2, it will print 2 at console and then print 3 in the console withing the function.

Correct Answer :

1 -> 2

console will display 1 because num += 1 is equivalent to num = num + 1 which means num = 1
main stack is finished and increment() will get executed when the num become 2
