Random Hex Colours - Question 10
Generate Random Hex Colour Values. Use a function that can return a full 6 digit hex value with one line.

function colour(){ return '#'; }
console.log(colour());

Correct Answer :

function colour(){ return '#' + Math.random().toString(16).subString(2,8) }
console.log(colour());

let c = Math.random();
console.log(c); //will get integer number
c = c.toString(16); //the radix 16 is the base used
console.log(c); //will get hexadecimal value
subString(2,8) // will return 6 digits
