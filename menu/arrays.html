Create an Array that is prepopulated with 5 random numbers

console.log('1', a1);

let a2

console.log('2', a2);

Correct Answer :

let a1 = Array.from ({ length: 5 }, n => Math.random())

console.log('1', a1)


The Array.from() static method creates a new, shallow-copied Array instance from an array-like or iterable object.
console.log(Array.from([1, 2, 3], x => x + x));
// expected output: Array [2, 4, 6]

let a2 = new Array(5).fill(1).map(n => n * Math.random());

console.log('2', a2)


The fill() method changes all elements in an array to a static value, from a start index (default 0) to an end index. (value, 시작위치, 종료위치 (종료위치 포함안됨))

const array1 = [1, 2, 3, 4];

// fill with 0 from position 2 until position 4
console.log(array1.fill(0, 2, 4));
// expected output: [1, 2, 0, 0]

// fill with 5 from position 1
console.log(array1.fill(5, 1));
// expected output: [1, 5, 5, 5]

console.log(array1.fill(6));
// expected output: [6, 6, 6, 6]
