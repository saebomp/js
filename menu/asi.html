ASI - Question 5
What happens when you run this code? why? how would you fix it?

let a = { a : 123 }

[a].forEach(function (x) { console.log(x) })

My Answer :

a.forEach(function (x) { console.log(x) })

Correct Answer :

let a = { a : 123 };

[a].forEach(function (x) { console.log(x) })

Bracket notation is more expressive than dot notation because it allows a variable to specify all or part of the property name. (in this case put the a object inside an array)

[a] doesn't know what a is before put semicolon after the object a.
